# 🧮 Simple Calculator

This is a basic calculator project built using **HTML, CSS, and JavaScript**.  
It performs simple arithmetic operations and is designed as a beginner-friendly project to understand the fundamentals of web development.

---

## 🚀 Features
- ➕ Addition  
- ➖ Subtraction  
- ✖️ Multiplication  
- ➗ Division  
- 🧹 Clear screen function  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

---

## 📂 Project Structure
calculator/
│── index.html
│── style.css
│── script.js

---

## ▶️ How to Run
1. Download or clone this repository  
2. Open `index.html` in your browser  
3. Start using the calculator
